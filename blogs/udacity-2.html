Udacity Blog Part Two: Design Concepts

What is grid based design?

One of the main uses for a grid based design is to basically arranging your elements in order and algined keeping your page concept neat and managable as grids encourages form. By utlizing a grid system, you're creating a set concept and way of organising all your elements promoting seamless layout.

Whats all the hype about responsive webpages?

Responsive website design ensures users have a good viewing experience no matter what type of device they use. Today Google prioritizes mobile-friendly sites in its search results algorithm. Using HTML and CSS to resize, hide, shrink, enlarge, or move the content to make it look good on on a variety of devices and window or screen sizes. Recent work also considers the viewer proximity as part of the viewing context as an extension for RWD.

What is semantic structure?

Semantic structure is a fancy term for an organization that represents meaning. For example, an English sentence is a semantic structure. Consider the following sentence structure: subject - verb - object.
