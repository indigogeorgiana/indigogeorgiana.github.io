What 'scope' is, and how it works in JavaScript.

The simplist explanation that www.w3schools.com came up with was "Scope determines the accessibility (visibility) of variables."" JavaScript have two main types of scope, Local scope and Global scope. Understanding JavaScript scope is essential to writing great code, learning and to become a better developer. UnderstandING where variables/functions are place and better accessible change our code making it better and faster to debug as well as excute.

What is local scope?

Local scope can only be accessed within the function between the curly braces and will only excute from within this function. Any function defined within another function has a local scope which is linked to the outer function. So it goes without saying that if we choose to define a function and create variables inside it, those variables then become locally scoped.

What is Global scope?

Global scope means your variables are visible globally. This means that everywhere within in your code. Global scope is usually placed between script tages.
