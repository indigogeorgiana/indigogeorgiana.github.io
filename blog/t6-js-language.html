JavaScript Language - Technical Blog

In the JavaScript language, describe all the use cases for each of the below:

parentheses ()

Parentheses have many different functions within JavaScript. They are generally used to contain a list of syntax passed, to control the order of execution. In the example below grouping, expressions and parameters are utlized in many different ways. The parameter list contains the information that is passed to the method to let the method do its job. A pair of parens at the end of a name is an easy way of identifying a method.

Example:

int a; a = (4 + 3) * 2; // Grouping expressions if (a > 10) { // Containing expressions line(a, 0, a, 100); // Containing a list of parameters }


brackets []

Square brackets are used to access elements and strings in arrays. Brackets [] are typically mostly used for accessing the properties of an object.

Example:

int nums[] = {1, 2, 3}; This line declares an array named nums[] and initializes its first three elements to values 1, 2, and 3.


braces {}

Curly braces, as they are known are used with many different elements classes, functions, loops, and conditionals. They are also used to mark the start and end of a code block. They can also be used to enclose a list that is used as an array initializer. Curly braces are used to define the beginning and end of functions blocks and statement blocks such as the for and if structures.

Example:

void draw() { for (int i=0; i < a.length; i++) { line(0, a[i], 50, a[i]); } }


single quotes ''

Upon reasearch I have found that with JavaScript most people favour single quotes over double quotes. An easy way to get around isolated cases of potentially broken strings is to use the opposite string syntax of the one you're currently using.

Example:Three ways to write a string with quotes:

single quotes (' ') alert("It's \"game\" time."); or alert('It\'s "game" time.');
