Use this example as a guide to getting your new Skeleton-based site up and running.

  • Do try to write your HTML from scratch though:
    • It's good practice
    • It's more satisfying
    • It builds 'muscle memory'

Keep it really simple until you've mastered the basic structure of each page using the grid system.

Remember: everything goes inside class="container". Columns (class="twelve columns") go inside rows (class="row").

There's a simple custom.css file here if you need an example to get you started.
