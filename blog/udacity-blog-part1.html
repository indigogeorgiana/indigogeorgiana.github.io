Udacity Blog Part One: HTML, CSS and the DOM


Explaining HTML, CSS and DOM to a non-technical friend?
  • HTML - Known as Hypertext markup language, it is a programming language used for the creating of web pages on the internet. HTML uses unique tags to let the browser know how a webpage should look and display online. These tags are designed to be used as pairs, and there are many different tags. HTML can also be used to add meta information to a web page but is not usually displayed by web browsers. There are many other similar languages such as XHTML which is quite similar but more strict version.
  • CSS - Cascading Style Sheets is used to format the layout of web pages on the internet and is a way to the design and look of HTML or XHTML web pages. The language was designed by the World Wide Web Consortium and is supported well by most modern web browsers. One advantage to using CSS is a web page can still be displayed and understood, even if the CSS is not working or removed.
  • DOM - A Document Object Model (DOM) is an application programming interface (API) used for valid HTML. With the DOM, programmers can build websites, add, modify or delete lements and content. Languages like HTML, CSS, and XHTML can be accessed, altered or using the DOM.

What is meant by boxifying design?

    Everything on a website is actually a box wow...that's amazeballs!! Simply put boxifying design is a method of taking everything needed to build a webpage and dividing them into various boxes. Boxifying all content of your webiste is quite a simple method. Before coding your website think first about everything that will be on your website. Grabbing a paper and pencil you can make space for images, text, headers, links and other content.

    Some the benefits to using the boxifying design concept are:

  • Makes the developers job easier when he/she is decided where everything is going to go on a web page.
  • Breaks down a large task into small tasks making it easier for a large team to manage.
