Technical Blog: Design to the Web


What a responsive site is, and why responsiveness is important?

Simply put, a responsive site is an internet website that is coded and designed so that it can be be displayed on any size screen. Because of a technology boost resulting in the increased sale of smart phones more people are using these devices to access the internet. Thus demanding the greater need for responsive websites.

Responsive websites are important because they are more flexible and can respond to resizing correctly. Older websites cannot do this as fonts, graphics and content drop off resulting in unhappy visitors. For web developers and designers this has removed the need for creating two separate websites as more thought put into the design against all platforms.

What a wireframe is and why we use it?

A wireframe is a rough draught or basic layout plan of a webpage, or what it might look like. Kind of like a skeletal three-dimensional model in which only lines are presented. An image or set of images which are used for planning a site's structure and functionality.

What mobile first design is, and why it's important?

Mobile first design is the process of planning from the smallest screen first, then later working our way up in screen size and the process becomes easier. This strategy works well because a smaller device is usually most difficult to design for. Also from a usability perspective planning for a smaller screen first is the most logical way forward when planning all content to display on screen.
